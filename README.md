# same-site-script #bugbounty #samesitescript
same site script scanner

Install
* git clone https://github.com/memmedrehimzade/same-site-script 
* cd same-site-script
* chmod +x ssitescan.sh

* ./ssitescan.sh target.tld  subdomains.txt
